# Лабораторна робота №3: Візуалізація даних

## Опис
Мета роботи — здійснити Data Cleaning та побудувати 5-8 інформативних графіків (включно з багатовимірними) за допомогою бібліотек `matplotlib` та `seaborn`.

## Датасет
Використовується [Automobile Dataset](https://archive.ics.uci.edu/dataset/10/automobile) з репозиторію UCI. 
Він повністю відповідає вимогам: Multivariate, містить Categorical, Integer та Real атрибути, а також має пропущені значення (Missing Values).

## Запуск
1. Встановіть залежності: `pip install -r requirements.txt`
2. Запустіть Jupyter Notebook: `jupyter notebook`
3. Відкрийте файл `visualization.ipynb`. Датасет завантажиться автоматично під час виконання коду.
