# Лабораторна робота №6: Обробка відео з камери (C++ та OpenCV)

## Опис проєкту
Програма зчитує відеопотік із веб-камери в реальному часі та дозволяє застосовувати різні фільтри обробки зображень за допомогою бібліотеки OpenCV. Проєкт має модульну архітектуру (кожен модуль винесено в окремий клас).

## Вимоги до системи та заліза
- **Операційна система:** Linux (рекомендується Ubuntu 20.04/22.04) або Windows Subsystem for Linux (WSL).
- **Інструменти збірки:** CMake (3.10+) та Make.
- **Бібліотеки:** OpenCV (`libopencv-dev`).

## Інструкція зі збірки та запуску

Проєкт містить зручні bash-скрипти для автоматизації розгортання.

1. **Встановлення залежностей:**
   Скрипт встановить OpenCV, CMake та компілятор (потребує прав `sudo`):
   ./preinstall.sh
2. **Компіляція проєкту:**
Скрипт створить папку build, налаштує CMake та збере виконуваний файл:
    ./build.sh
3. **Запуск програми:**
Скрипт запустить готовий бінарний файл:
    ./run.sh
## Керування (Інтерфейс)
Після запуску відкриється вікно з трансляцією відео. Використовуйте клавіатуру для перемикання режимів обробки:

1 — Нормальний режим (без змін)

2 — Інверсія кольорів

3 — Розмиття (Gaussian Blur)

4 — Виділення контурів (Canny)

5 — Фільтр Собеля

q або ESC — Вийти з програми
