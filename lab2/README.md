## Вимоги до системи та середовища
- **Інтерпретатор:** Python 3.9 або вище.
- **Операційна система:** Windows / macOS / Linux.

## Налаштування середовища та встановлення залежностей
1. Відкрийте термінал у папці проєкту.
2. Створіть віртуальне середовище:
   `python -m venv venv`
3. Активуйте віртуальне середовище:
   - Для Windows: `venv\Scripts\activate`
   - Для macOS/Linux: `source venv/bin/activate`
4. Встановіть необхідні залежності з файлу requirements:
   `pip install -r requirements.txt`

## Інструкція щодо отримання датасетів
- **Частина 1 (Дані NOAA):** Завантажуються автоматично за допомогою модуля `urllib` під час виконання скрипту.
- **Частина 2 (Споживання електроенергії):** Необхідно завантажити вручну (оскільки файл дуже великий). 
  1. Перейдіть за посиланням: [Household Power Consumption Dataset](https://archive.ics.uci.edu/ml/machine-learning-databases/00235/household_power_consumption.zip)
  2. Завантажте архів.
  3. Розпакуйте файл `household_power_consumption.txt` та покладіть його в корінь папки `lab2`.

## Інструкція для запуску скрипту
1. Переконайтеся, що віртуальне середовище активоване.
2. Запустіть Jupyter Notebook командою:
   `jupyter notebook`
3. У браузері відкрийте файл `data_analysis.ipynb` для перевірки першої частини. Запустіть усі комірки послідовно.
4. Відкрийте файл `power_consumption.ipynb` для перевірки другої частини та запустіть усі комірки.
