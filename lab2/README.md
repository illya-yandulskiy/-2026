# Лабораторна робота №2 Наука про дані: підготовчий етап
## Опис завдання 
Робота присвячена аналізу даних NOAA (VHI-індекс) та аналізу споживання електроенергії.
## Вимоги до середовища
- Python 3.11.9
- Бібліотеки: pandas, requests, jupyter
- Встановлення: pip install -r requirements.txt
## Як запустити
1. Створіть віртуальне середовище: python -m venv venv
2. Активуйте його та встановіть залежності.
3. Запустіть Jupyter Notebook: jupyter notebook
4. Відкрийте `data_analysis.ipynb` та `power_consumption.ipynb`.
