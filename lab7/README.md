# Лабораторна робота №7: Інтеграція OpenCV DNN та Багатопотоковості

## Мета роботи
Поєднання класичного програмування на C++ із сучасним CV/ML. Реалізація детекції об'єктів (облич) за допомогою `cv::dnn` (Deep Neural Networks) та вирішення проблеми блокування інтерфейсу важкими обчисленнями за допомогою багатопотоковості (`std::thread`, `std::mutex`, `std::condition_variable`).

## Інструкція зі збірки та запуску

Проєкт автоматизовано за допомогою bash-скриптів. Для запуску виконайте наступні команди в терміналі Linux:

1. Надайте права на виконання скриптам:
   chmod +x *.sh
2. Встановіть залежності та завантажте моделі нейромережі:
    ./preinstall.sh
3. Зберіть проєкт:
    ./build.sh
4. Запустіть програму:
    ./run.sh
## Керування (Гарячі клавіші)
1 - Нормальний режим (без фільтрів)

2 - Інверсія кольорів

3 - Розмиття (Gaussian Blur)

4 - Виділення контурів (Canny)

5 - Фільтр Собеля

F або f - Детекція облич (DNN) у фоновому потоці

Q, q або Esc - Вихід
