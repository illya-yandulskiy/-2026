# Лабораторна робота №5: Створення веб-додатку за допомогою Streamlit

## Опис
Цей проєкт є інтерактивним веб-додатком, створеним за допомогою бібліотеки Streamlit. Він дозволяє користувачам візуалізувати та аналізувати часові ряди вегетаційних індексів (VCI, TCI, VHI) на основі даних NOAA для різних областей України.

## Функціонал
- Вибір індексу (VCI, TCI, VHI) та області через випадаючі списки (Dropdowns).
- Фільтрація даних за роками та тижнями за допомогою повзунків (Sliders).
- Сортування відфільтрованих даних за зростанням або спаданням (Checkboxes).
- Відображення результатів у трьох вкладках (Tabs):
  1. Таблиця з відфільтрованими даними.
  2. Графік динаміки обраного індексу для вибраної області.
  3. Стовпчаста діаграма для порівняння середнього значення індексу по всіх областях.
- Кнопка швидкого скидання всіх налаштувань до початкових значень (Reset).
