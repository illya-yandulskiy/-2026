# Лабораторна робота №4: Інтерактивна візуалізація та фільтрація сигналів

## Опис
Програма дозволяє користувачам малювати графік функції гармоніки з накладеним шумом. Реалізовано інтерактивний інтерфейс (слайдери, чекбокси, кнопки) для зміни параметрів гармоніки та шуму в реальному часі. Також реалізовано низькочастотний фільтр Баттерворта для відсікання шуму.

## Вимоги до системи та середовища
- **Інтерпретатор:** Python 3.9 або вище.
- **ОС:** Windows / macOS / Linux.

## Налаштування середовища
1. Створіть та активуйте віртуальне середовище:
   `python -m venv venv`
   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`
2. Встановіть необхідні залежності:
   `pip install -r requirements.txt`

## Інструкція для запуску
1. Активуйте віртуальне середовище.
2. Запустіть Jupyter Notebook командою `jupyter notebook` або відкрийте файл у VS Code.
3. Відкрийте файл `lab4.ipynb`.
4. Запустіть усі комірки по черзі.
